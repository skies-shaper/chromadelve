Chromadelve

A roguelike, dungeon delving, color-coded game of skill and luck and there's a turtle in here somewhere too.

Game content :)

Controls:

  • 'm': toggle map
  • 'r': generate new map (for testing purposes only. Will be deprecated soon.)
  • Arrow Keys to move
  • 't': opens the chat box
